<h1>TRATAMENTO DE EXCEÇÕES</h1>
<p>Exercício didático para aprendizado do tratamento de exceções em java
</p>
<p></strong>O programa passará por atualizações, a princípio estamos tratando a possível exceção dentro do programa principal para entender a forma correta e ir implementando as boas práticas.</p>
